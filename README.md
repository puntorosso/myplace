# MyPlace

Landing della piattaforma **MyPlace** — aiuta chi si avvicina alla pensione a vivere bene in borghi accessibili dell'entroterra spagnolo, dentro una **comunità internazionale di persone affini**. La prima comunità (artisti e professionisti dello spettacolo) è in cammino verso **Larva** (Jaén, Andalusia).

## Pagine
- `index.html` — landing della piattaforma (multilingue **IT / ES / EN / DE**)
- `larva.html` — pagina della comunità "Artisti → Larva", con galleria foto
- `privacy.html` — informativa privacy (bozza GDPR)

## Caratteristiche
- Multilingua con selettore di lingua (i18n lato client, scelta memorizzata)
- Design responsive, accessibile, senza dipendenze esterne (solo Google Fonts)
- Modulo di contatto che apre un'email precompilata verso il referente
- SEO + Open Graph + favicon + sitemap/robots
- Foto di Larva da Wikimedia Commons (licenza libera) — da sostituire con foto autentiche

## Pubblicazione
Sito statico ospitato su GitHub Pages. Nessun backend.

### Da fare prima del lancio “vero”
- Sostituire le foto con immagini autentiche del paese e della comunità
- Collegare il modulo a un servizio di raccolta (es. Formspree / Google Form) o a un backend
- Validare l'informativa privacy con un legale
- Eventuale dominio personalizzato
